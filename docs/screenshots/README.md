# Screenshots

These images are referenced from the top-level [`README.md`](../../README.md)
"Screenshots" section. To refresh them, drop replacement PNGs in this folder
using the **exact filenames** below — the README links to them by name.

| Filename | What it shows | Suggested capture |
|---|---|---|
| `game-session.png` | A live DragonRealms session — game stream, Room panel, Backpack/inventory, and the Logons/Talk/Whispers/Thoughts/Combat stream tabs, with the vitals bar. | Connected, standing in a well-described room (e.g. Riverhaven, Town Hall Road). Whole window. |
| `mapper.png` | The Mapper panel — zone picker, Edit/Record/New/Save/Labels toolbar, level controls, and the map canvas. | Mapper window with a zone loaded (or the "No zone loaded" empty state). |
| `connect-dialog.png` | The **Connect to DragonRealms** dialog — Profile, Mode, Instance, Account/Password, Character fetch, and the TLS toggle. | File → Connect…, before entering credentials (leave Account/Password blank). |

## Tips

- **Crop to the window**, not the whole desktop — no taskbar or wallpaper.
- **Don't capture credentials.** Leave Account/Password empty in the connect
  dialog, and avoid showing a real character name in the title bar where it's
  easy to redact.
- PNG, captured at 100% display scaling. ~1000–2000 px wide is plenty; GitHub
  scales them down in the rendered README.
